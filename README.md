# Rebar Mobs

A [Rebar](https://github.com/pylonmc/rebar) addon that add skulls for almost all living entities, allowing players to capture entities, and make them as inventory companion.

## Requirement

- [Rebar](https://github.com/pylonmc/rebar) 0.35.0+
- [Pylon](https://github.com/pylonmc/pylon)
